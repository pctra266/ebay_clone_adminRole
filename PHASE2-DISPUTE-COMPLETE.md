## ?? BUSINESS LOGIC HIGHLIGHTS

### **?? Money Flow in ResolveDisputeCommand:**

**Buyer Wins:**
```csharp
sellerWallet.DisputedBalance -= amount;      // Remove from frozen
sellerWallet.TotalRefunded += amount;         // Track refund
CreateFinancialTransaction(buyer, amount);    // Refund to buyer
```

**Seller Wins:**
```csharp
sellerWallet.DisputedBalance -= amount;      // Remove from frozen
sellerWallet.AvailableBalance += amount;     // Release to seller
```

**Split Decision:**
```csharp
sellerWallet.DisputedBalance -= disputedAmount;
sellerWallet.AvailableBalance += sellerKeeps;
sellerWallet.TotalRefunded += refundAmount;
CreateFinancialTransaction(buyer, refundAmount);
```

### **?? Statistics Calculation:**

- **Average Resolution Time:** `(ResolvedAt - CreatedAt).TotalHours`
- **Average Response Time:** `(FirstResponseAt - CreatedAt).TotalHours`
- **Buyer Win Rate:** `BuyerWins / TotalResolutions * 100`
- **Urgent Cases:** `Deadline <= Now + 24 hours`

### **?? Priority Indicators:**

```
?? Critical: Deadline < 24h + Amount > $1000
?? High: Deadline < 48h OR Amount > $500
?? Medium: Standard cases
? Low: Low value, long deadline
```
