### **Dispute Entity - Enhanced Fields:**

```csharp
// BASIC (existing)
? Id, OrderId, RaisedBy, Description, Status, Resolution

// CLASSIFICATION (new)
? Type                // 'INR', 'INAD', 'Damaged', 'Counterfeit'
? Subcategory         // 'WrongSize', 'WrongColor', etc.
? DesiredOutcome      // 'FullRefund', 'PartialRefund', 'Replacement'

// EVIDENCE (enhanced)
? BuyerEvidence       // JSON array of images/videos

? BuyerResponse       // Counter-arguments
? ChatLog             // Full conversation

// NEGOTIATION (new)
? OfferHistory        // JSON: [{amount, offeredBy, status}]
? NegotiationRounds   // Count of back-and-forth
? LastOfferAmount     // Latest offer amount

// TIMELINE (new)
? CreatedAt           // When case opened
? FirstResponseAt     // Seller's first response
? EscalatedAt         // When moved to admin

// ADMIN ASSIGNMENT (new)
? AssignedTo          // Admin handling this case
? AssignedAt          // Assignment timestamp
? ViewCount           // How many times viewed
? LastViewedAt        // Last view timestamp

// RESOLUTION (enhanced)
? AdminNotes          // Admin's private notes
? ResolvedBy          // Admin ID
? ResolvedAt          // Resolution timestamp
? Winner              // 'Buyer', 'Seller', 'Split'

// REFUND PROCESSING (new)
? RefundAmount        // Final refund amount
? RefundMethod        // 'OriginalPayment', 'StoreCredit'
? RefundProcessedAt   // When money transferred
? RefundTransactionId // Payment gateway ID

// SHIPPING (new)
? TrackingNumber      // Original shipment
? DeliveryStatus      // 'Delivered', 'Lost', etc.
? RequiresReturn      // Need item back?
? ReturnTrackingNumber // Return shipping
? ReturnReceivedAt    // When item returned

// FLAGS (new)
? IsAutoGenerated     // System-initiated
? IsHighValue         // Amount > $1000
? IsVeRO              // Brand owner complaint
```
